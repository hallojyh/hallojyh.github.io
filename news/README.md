# 新闻管理

每条新闻一个文件夹，正文和图片放在一起。

## 添加新新闻（2 步）

### 1. 创建文件夹

在 `news/` 下新建文件夹，命名建议：`YYYY-MM-DD-标题关键词`：

```
news/
├── 2026-05-30/               ← 新建的文件夹
│   ├── content.json          ← 正文（必须）
│   └── cover.jpg             ← 封面图（可选）
└── news-data.js              ← 登记信息
```

### 2. 登记到 news-data.js

打开 `news/news-data.js`，在数组最前面插入：

```js
{
    date: '2026-06-15',
    tag: '公司新闻',
    title: '这里写新闻标题',
    desc: '这里写一句话摘要。',
    icon: '📰',
    img: 'news/2026-06-15/cover.jpg',   // 可选：封面图
    folder: 'news/2026-06-15'            // 指向文件夹
}
```

## content.json 格式

```json
{
  "paragraphs": [
    "正文第一段。",
    "正文第二段。",
    "正文第三段。"
  ]
}
```

每个数组元素渲染为一个 `<p>` 段落。

## 图片放哪里

封面图、正文配图全部放在该新闻的文件夹内：

```
news/2026-06-15/
├── content.json
├── cover.jpg        ← 封面：引用路径 news/2026-06-15/cover.jpg
└── demo.jpg         ← 正文配图（目前需手动在 content.json 段落中写 HTML）
```

## 文件夹示例

已有 `news/2026-05-30/` 作为参考模板，新新闻直接复制重命名。

## 排序规则

`news-data.js` 数组第 1 条 → 置顶大卡片，其余 3 列排列。
