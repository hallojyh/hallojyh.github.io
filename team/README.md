# 团队成员管理

每个成员一个文件夹，介绍内容和照片放在一起。卡片由 JS 自动渲染。

## 添加新成员（2 步）

### 1. 创建文件夹

在 `team/` 下新建文件夹，命名建议用姓名拼音：

```
team/
├── li-ming/                  ← 新建的文件夹
│   ├── content.json          ← 详细介绍（必须）
│   └── avatar.jpg            ← 照片（可选）
└── team-data.js              ← 登记信息
```

### 2. 登记到 team-data.js

打开 `team/team-data.js`，在数组末尾插入：

```js
{
    id: 'li-ming',                // 唯一标识，姓名拼音
    name: '李明',                  // 姓名
    title: '算法工程师',           // 职位
    avatarInitial: '李',           // 头像文字，无照片时显示
    avatarGradient: 'grad-edu',    // 头像渐变色（四选一）
    shortBio: '简介1-2句话。',     // 列表页显示的简短介绍
    photo: 'team/li-ming/avatar.jpg',  // 照片路径，可选
    folder: 'team/li-ming'         // 指向文件夹
}
```

## content.json 格式

```json
{
  "sections": [
    { "heading": "个人简介", "content": "段落内容..." },
    { "heading": "教育背景", "content": "段落内容..." },
    { "heading": "工作经历", "content": "段落内容..." },
    { "heading": "技术专长", "content": "段落内容..." },
    { "heading": "个人理念", "content": "段落内容..." }
  ]
}
```

每个 section 渲染为一个标题+段落板块。板块数量不限，可按需增减。

> ⚠️ JSON 中请使用纯文本，不要出现弯引号 `""`、反斜杠转义 `\"` 等特殊字符。

## 添加照片

把照片（jpg/png/webp）放入对应成员文件夹，在 `team-data.js` 中加一行 `photo` 即可。

照片将自动裁剪为圆形，同时显示在列表卡片和详情页。不加 `photo` 字段时自动回退为渐变色头像。

## 删除成员

从 `team-data.js` 数组中删除对应条目即可，卡片自动消失。可选择性删除 `team/xxx/` 文件夹。

## 渐变色速查

| 值 | 效果 | 适合角色 |
|---|---|---|
| `grad-edu` | 紫→蓝渐变 | 算法/AI/教育 |
| `grad-industry` | 青→蓝渐变 | 硬件/工程/工业 |
| `grad-service` | 绿→青渐变 | 产品/运营/服务 |
| `grad-hero` | 蓝→紫→青渐变 | 创始人/管理 |

---

> 无需修改 `index.html`，所有卡片由 JS 根据 `team-data.js` 自动渲染。
